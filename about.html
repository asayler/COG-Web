About

This is the web front-end for the COG (Computerized Online Grading) System. COG is a system for providing rapid feedback on programming assignments and for automatically assessing assignment submission grades. As of the Fall 2015 semester, COG has been in beta use for over a year and has graded thousands of studunt submissions.

This front-end and the associated COG backend are both licensed under the AGPLv3, the full text of which can be found in the associated COPYING file. The source code is available via GitHub. COG-Web uses the Bootstrap web framework, provided under the terms of the MIT license. It also uses the jQuery JavaScript library, provided under the terms of the MIT license.

You may also be interested in the COG API and backend (where all the magic happens). There is also a CLI front-end that may be used as an alternative to this web front-end.
